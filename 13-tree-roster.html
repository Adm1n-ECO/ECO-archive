How the AI search works The search box calls the Claude API with the user's free-text description. The prompt instructs Claude to act as a botanist and return 3–5 real tree species that match — each with common name, scientific name, native region, one defining visual quality, and one cultural/symbolic quality. The response is structured JSON, rendered as the species cards above. This makes every user's tree discovery feel personal and curated, not like a dropdown menu.

Relationship cross-linking The "Also related to" field creates cross-references in the graph database — not just "Vikas's aunt" but "sister of Ramesh" and "mother of Raj." These cross-links are what make the tree intelligent. When Raj joins independently and builds his own tree, the system can suggest: "We think you may already be connected to the Bakshi household — Sunita Mehta appears in both trees." This is the consent-based merge experience.
