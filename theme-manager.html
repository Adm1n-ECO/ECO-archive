Implementation Guide

Including in every page

<!-- In <head> -->
<link rel="stylesheet" href="css/theme.css">

<!-- In <body> before </body> -->
<a href="/admin" class="admin-float-btn" title="Admin">⚙</a>
<script src="scripts/theme-selector.js"></script>

Admin button activation

Call from your auth module after verifying the admin session:

// Show admin button (theme selector moves up 40px automatically)
ECOTheme.setAdminVisible(true);

// Hide admin button
ECOTheme.setAdminVisible(false);

Using theme tokens in CSS

/* Always use CSS variables — never hardcode hex values */
.my-section {
  background: var(--color-bg);    /* matches current theme bg */
  color: var(--color-text);       /* always high-contrast */
  border: 1px solid var(--color-border);
  font-size: var(--text-sm);      /* minimum 14px */
}

/* Accents */
.highlight { color: var(--color-accent-c); }  /* ECO Cyan */
.success   { color: var(--color-accent-g); }  /* ECO Green */
.warning   { color: var(--color-accent-a); }  /* Admin Amber */

Theme selector position

The floating theme selector is fixed at bottom: 20px; right: 16px by default. When the admin button is visible (44px tall, also at bottom: 20px), the theme selector automatically moves up to bottom: 104px (20 + 44 + 40px buffer), keeping a 40px gap above the admin button.
